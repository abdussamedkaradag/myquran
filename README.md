# MyQuran Projesi

Modern ve kullanıcı dostu bir Kuran-ı Kerim arayüzü.

## Özellikler

- Sure ve ayet arama
- Türkçe meal okuma
- Sureler listesi
- Ayet kopyalama
- Modern ve responsive tasarım

## Teknolojiler

- Next.js
- TypeScript
- Tailwind CSS
- React

## Geliştirme

Projeyi yerel ortamınızda çalıştırmak için:

```bash
npm install
npm run dev
```

Tarayıcınızda `http://localhost:3000` adresini açın.
